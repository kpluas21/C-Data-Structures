Field Documentation

◆ num_of_edges

GraphC::num_of_edges

The number of edges contained in our graph. A single edge is defined as a connection between a pair of vertices.

◆ num_of_vertices

GraphC::num_of_vertices

The number of vertices contained in our graph. A single vertex is defined as a data node or point that make up a graph.

◆ vertices

GraphC::vertices

A pointer to the first vertex in our graph. Every unique instance of a vertex is stored in a contiguous array. It's easier to think of the graph as an array of linked lists.
