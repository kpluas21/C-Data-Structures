This is \mbox{\hyperlink{structGraphC}{GraphC}}, an undirected, unweighted graph. This graph will be represented using an adjacency list through the use of linked lists.

\mbox{\hyperlink{structGraphC}{GraphC}} stores a contiguous array of Vertexs which serve as heads to a linked list of adjacent neighbors.
