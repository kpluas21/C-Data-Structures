Detailed Description

Specification of DynamiC.

Author
Kevin Pluas (kplua.nosp@m.s21@.nosp@m.gmail.nosp@m..com)
Version
0.1
Date
2023-04-02

This structure uses void pointers and regular arrays. It handles ints, chars, doubles and floats. Strings are to be implemented soonish... Everything you need should be in this one header file. Simply include it and call the _init function. Make sure to include what type you are gonna use the array for. This was not tested for every outcome yet.

Because this was created using LOTS of void pointers, expect to see and use a lot of casting, again, it should work if you're casting appropriately.

DynamiCs are zero-based and are designed with fast insertion in mind. Appending should occur in constant time unless capacity was reached in which case it will automatically resize and attempt to append the element again.
